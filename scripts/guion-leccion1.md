## Desarrollo (6 min)

### ¿Qué es programar?

Programar es darle instrucciones a una computadora para que haga algo. Como una receta de cocina: paso a paso, la computadora sigue cada instrucción exactamente como se la diste.

### ¿Qué es Java?

Java es uno de los lenguajes de programación más usados del mundo. Se usa para crear:
- Aplicaciones de Android
- Videojuegos (como Minecraft)
- Sistemas bancarios
- Aplicaciones web

### Tu primer programa en Java

Todo programa Java tiene esta estructura mínima:

```java
public class MiPrograma {
    public static void main(String[] args) {
        // aquí van las instrucciones
    }
}
```

- `public class MiPrograma` — define el programa. El nombre debe coincidir con el nombre del archivo.
- `public static void main(String[] args)` — es el punto de entrada. Todo programa Java empieza aquí.
- Las llaves `{ }` agrupan las instrucciones.

### Imprimir en pantalla

```java
System.out.println("Hola Mundo");
```

`System.out.println()` imprime texto en la consola y salta a la siguiente línea. Todo lo que quieras mostrar va entre las comillas.
