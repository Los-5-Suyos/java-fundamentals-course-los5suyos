## Desarrollo (6 min)

### ¿Qué es programar?

Programar es darle instrucciones a una computadora para que realice una tarea.

Piensa en una receta de cocina.

Si quieres preparar una pizza, necesitas seguir pasos específicos:

Preparar la masa.
Agregar los ingredientes.
Hornearla.
Servirla.

La computadora funciona de manera similar. Nosotros escribimos instrucciones y ella las ejecuta exactamente en el orden indicado.

Por eso, cuando programamos, debemos ser claros y precisos.

<br>

### ¿Qué es Java?

 Java es uno de los lenguajes de programación más populares y utilizados en el mundo.

Con Java se pueden crear muchas cosas interesantes, por ejemplo:

Aplicaciones para Android.
Videojuegos como Minecraft.
Sistemas bancarios.
Aplicaciones web.
Programas empresariales utilizados por millones de personas.

Java es una excelente opción para comenzar a aprender programación porque es fácil de leer y ayuda a desarrollar buenas prácticas desde el inicio.


<br>

### Tu primer programa en Java

Todo programa en Java tiene esta estructura básica:

```java
public class MiPrograma {
    public static void main(String[] args) {
        // aquí van las instrucciones
    }
}
```

Al principio puede parecer un poco extraño, pero no te preocupes.

Por ahora solo necesitas conocer tres partes importantes:

<br>

### 1. La clase

```java
public class MiPrograma
```

Aquí se define el nombre del programa.

El nombre de la clase debe coincidir con el nombre del archivo.

<br>

### 2. El método principal

```java
public static void main(String[] args)
```

Esta línea le indica a Java dónde debe comenzar a ejecutar el programa.

No es necesario memorizarla ni entender cada palabra todavía. Más adelante aprenderemos qué significa cada parte.

<br>

### 3. Las llaves

```java
{
}
```

Las llaves agrupan las instrucciones del programa.

Todo lo que queremos que Java ejecute debe estar dentro de ellas.

<br>

### Imprimir mensajes en pantalla

La instrucción más sencilla para mostrar información es:

```java
System.out.println("Hola Mundo");
```
Cuando ejecutamos el programa, veremos este mensaje en la consola.

La computadora mostrará exactamente el texto que coloquemos entre las comillas.

**Por ejemplo:**

```java
System.out.println("Me gusta aprender Java");
```

<br>

**Resultado:**

> Me gusta aprender Java

Cada vez que usamos System.out.println(), Java muestra el texto y pasa a la siguiente línea.
